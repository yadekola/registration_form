# Registration Form

## Description
This project is a simple registration_ form created using HTML and CSS. It allows users to input their information and submit the form.

## Features
- Input fields for name, email, password, and confirm password
- Submit button to submit the form
- Basic styling using CSS

## Technologies Used
- HTML
- CSS

## Usage
1. Clone the repository to your local machine.
2. Open the index.html file in your preferred web browser.
3. Fill out the form fields with your information.
4. Click the "Submit" button to submit the form.
